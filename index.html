PROJECTS

Digiberkat: E-commerce System with Web Admin and Mobile App

A full-stack e-commerce platform to digitize Berkat Cell's (a brick-and-mortar store) operations. The system consists of a React-built mobile application for customers to browse and order, and a Laravel web-based admin dashboard for store management. A RESTful API developed using Golang powers the backend and serves data to both web and mobile clients, while JWT is implemented for a secure authentication system. Additionally, the system integrates AI-driven features into the mobile app by using a Python-based backend service.

Tech stack: Laravel (web frontend), React (mobile frontend), Golang (backend), Python (AI backend), MySQL (database)

Machine Learning Model for Urban Economic Status Classification

A predictive model built with Python that applies the k-Nearest Neighbors algorithm to automatically classify the economic tier of cities in Indonesia based on their financial metrics. The model was trained on a custom dataset which had undergone comprehensive preprocessing by cleaning and labeling. The trained model can successfully determine if the economic status of a city is "Low", "Middle", or "High" by Indonesian standards based on its financial inputs, with a classification accuracy of 75%.

Tech stack: Python, Scikit-learn, Pandas, NumPy
